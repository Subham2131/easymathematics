Essential Calculus Formulas

1. Differentiation Formulas

d/dx [c] = 0
d/dx [xⁿ] = n * xⁿ⁻¹
d/dx [eˣ] = eˣ
d/dx [ln(x)] = 1/x
d/dx [sin(x)] = cos(x)
d/dx [cos(x)] = -sin(x)
d/dx [tan(x)] = sec²(x)
d/dx [csc(x)] = -csc(x) * cot(x)
d/dx [sec(x)] = sec(x) * tan(x)
d/dx [cot(x)] = -csc²(x)
d/dx [u * v] = u' * v + u * v'
d/dx [u / v] = (u' * v - u * v') / v²

2. Integration Formulas

∫ c dx = c * x + C
∫ xⁿ dx = (xⁿ⁺¹ / (n+1)) + C (n ≠ -1)
∫ eˣ dx = eˣ + C
∫ 1/x dx = ln|x| + C
∫ sin(x) dx = -cos(x) + C
∫ cos(x) dx = sin(x) + C
∫ sec²(x) dx = tan(x) + C
∫ csc²(x) dx = -cot(x) + C
∫ sec(x) * tan(x) dx = sec(x) + C
∫ csc(x) * cot(x) dx = -csc(x) + C

3. Limit Formulas

lim (x → a) c = c
lim (x → a) x = a
lim (x → a) [xⁿ] = aⁿ
lim (x → a) [f(x) ± g(x)] = lim (x → a) f(x) ± lim (x → a) g(x)
lim (x → a) [f(x) * g(x)] = lim (x → a) f(x) * lim (x → a) g(x)
lim (x → a) [f(x) / g(x)] = lim (x → a) f(x) / lim (x → a) g(x) (g(a) ≠ 0)
lim (x → 0) sin(x)/x = 1
lim (x → 0) (1 + x)¹/ˣ = e

4. Miscellaneous Formulas

d/dx [aˣ] = aˣ * ln(a)
∫ aˣ dx = (aˣ / ln(a)) + C (a > 0, a ≠ 1)
d/dx [logₐ(x)] = 1 / (x * ln(a))
∫ logₐ(x) dx = x * (logₐ(x) - 1) + C
